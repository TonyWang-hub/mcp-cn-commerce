## 常见问题排查

### 订单问题

**订单查询为空**
```
我查询京东订单返回空列表，可能的原因：
1. 时间范围设置是否正确？
2. 订单状态筛选是否正确？
3. 是否有权限访问该店铺的订单？
```

**订单状态不更新**
```
订单状态显示为"待发货"，但实际已经发货了，可能的原因：
1. 平台数据同步延迟
2. 物流信息未更新
3. 需要手动刷新订单状态
```

### 商品问题

**商品库存不准确**
```
商品库存显示为 0，但实际有库存，可能的原因：
1. 库存数据同步延迟
2. 多仓库库存未合并
3. 预售商品库存单独计算
```

**商品价格不一致**
```
同一商品在不同平台价格不同，可能的原因：
1. 平台活动价格
2. 优惠券/满减
3. 运费差异
```

### 广告问题

**广告数据延迟**
```
广告报表数据不是实时的，可能的原因：
1. 平台数据处理延迟（通常 1-2 小时）
2. 归因窗口期
3. 数据校验中
```

**广告花费异常**
```
广告花费突然增加，可能的原因：
1. 竞价环境变化
2. 预算设置错误
3. 定向人群扩大
4. 素材质量下降导致 CPC 上升
```

## 最佳实践

### 查询优化

1. **限制时间范围**：查询时尽量指定时间范围，避免查询全量数据
2. **分页查询**：数据量大时使用分页，避免一次性加载过多数据
3. **筛选条件**：使用筛选条件减少返回数据量

### 数据分析

1. **对比分析**：多平台数据对比，找出差异和问题
2. **趋势分析**：关注数据变化趋势，而不是单点数据
3. **异常检测**：设置阈值，及时发现异常情况

### 安全建议

1. **权限控制**：只授予必要的 API 权限
2. **定期轮换**：定期更新 API 密钥
3. **监控日志**：定期检查 API 调用日志
