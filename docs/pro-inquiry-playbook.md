# Pro 咨询回复手册

> 用途：处理仓库里 `[Pro] 咨询` issue 的标准回复。每条 inbound 都当一次真实用户访谈，记录对方的真实痛点。

## 0. 原则

1. **先诚实划清"能/不能"，再接需求**——工具是**只读数据连接器**，不替客户做投流/选品策略；价值是"让 AI 基于真实数据做分析和优化建议，省掉翻后台"。假装能做策略，留不住人。
2. **平台覆盖度说实话**：抖店（千川/巨量）覆盖最完整；拼多多（多多进宝）、天猫（万相台）的投流数据目前覆盖较浅，排期中。
3. **把"怎么投"引导到"投得怎么样"**——前者是运营经验活（工具不解决），后者是数据分析活（工具的主场）。资格问题里务必问出对方痛点落在哪边。
4. **多店 + token 痛点**是 Pro 的天然卖点：千川 access token 24h 过期、多店聚合。

## 1. 资格问题（qualifying，先问清再深聊）

1. 主要投哪个平台的流？（抖店千川 / 拼多多 / 天猫万相台）
2. 投流最大的痛点：**不知道投哪/怎么投**，还是**投了不知道效果好不好、不知道调哪**？
3. 月投放预算量级？目标是拉新 / ROI / GMV？
4. 现在怎么看投流数据？（手动翻后台 / 有人做表 / 没人管）

> 痛点落在「不知道效果好不好、该调哪」= 强对口；落在「从零教我怎么投」= 工具帮不上，诚实说明。

>
> 你管 3 家店，Pro 的**多店聚合 + token 自动刷新**（千川 24h 过期那个痛点）正好对口。

> 想帮你更准，先问一句：你现在投流最大的痛点，是**不知道怎么投**，还是**投了不知道效果好不好、不知道调哪**？

## 3. 回复模板 B — 完整诚实版（对方偏技术/谨慎时用）

> 你好，感谢咨询 🙏 先说清楚这个项目能帮你什么、不能帮什么，免得浪费你时间：
>
> **它是什么**：一个**只读**的数据连接器，让 AI 直接读你店铺的经营/投流数据。对你的场景，最对口的是**抖店的千川/巨量投流数据**——AI 能帮你做这类事：
> - "这周哪些计划消耗最高但 ROAS 最低？该关哪些？"
> - "对比上月，我的整体 ROI 趋势怎么样？"
> - "哪些计划值得加预算？"
>
> **它不是什么**：它**不替你制定投流策略**（选品、定向、出价打法），那是运营经验的活；它做的是**把数据摆到 AI 面前，让 AI 帮你做分析和优化建议**，省掉你天天翻后台看报表。
>
> **关于你的 3 个平台**：抖店（千川）覆盖最完整；拼多多（多多进宝）、天猫（万相台）的投流数据目前覆盖较浅，正在排期。
>
> 想更精准帮你，问几个问题：
> 1. 主要投哪个平台的流？
> 2. 投流最大的痛点 —— **不知道投哪/怎么投**，还是**投了不知道效果好不好、不知道该调哪**？
> 3. 大概的月投放预算量级？目标是拉新、ROI 还是 GMV？
> 4. 现在你是怎么看投流数据的？
>
> 你管 3 家店，Pro 版的**多店聚合 + token 自动刷新**正好对口。等你回上面几个问题，我给你更具体的建议。

## 4. 第 2 轮 — 投流分析具体怎么用（对方答"投了不知道效果好不好"后）

> 那这工具就对口了，给你看具体能怎么用 👇
>
> 连上千川后，你直接用大白话问 AI，它在后台自动拉报表、不用你翻后台导表：
> - "列出本周消耗 Top10 的计划，按 ROAS 排序，标出 ROAS 低于 1.5 的"
> - "哪些计划这周消耗涨了但转化没涨？建议关停哪些？"
> - "对比上周，我整体的消耗、GMV、ROI 变化"
> - "账户余额还能投几天？"
>
> Pro 版额外帮你：
> ① 千川 token 24h 过期**自动续**（不用天天手动换）；
> ② 3 家店**统一管、跨店对比**。
>
> 要不要我先用**模拟数据**给你跑一遍效果看看？（不用你给任何授权，先看像不像你要的）确认有用，再上你真实的店。

> 备注：先用模拟数据 demo（`templates/` 里有），**不要一上来就要对方的真实授权**——降低戒心，也合规。
