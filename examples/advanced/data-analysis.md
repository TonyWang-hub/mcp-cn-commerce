## 销售分析

### 销售趋势分析

```
请帮我分析过去 30 天的销售趋势：
1. 每日订单数量变化
2. 每日销售额变化
3. 找出销售高峰和低谷
4. 分析可能的原因
```

### 销售季节性分析

```
请帮我分析销售的季节性特征：
1. 每周各天的销售分布
2. 每月各天的销售分布
3. 节假日对销售的影响
4. 促销活动对销售的影响
```

### 销售归因分析

```
请帮我分析销售的归因：
1. 各渠道的销售贡献
2. 各广告计划带来的销售
3. 自然流量 vs 付费流量的销售占比
```

### 客单价分析

```
请帮我分析各平台的客单价：
1. 各平台平均客单价
2. 客单价分布
3. 高客单价订单特征
4. 提升客单价的建议
```

## 分析报告模板

### 日常分析报告

```
请帮我生成日常分析报告：

1. 销售数据
   - 今日销售额
   - 今日订单数
   - 与昨日对比

2. 库存数据
   - 低库存预警
   - 滞销商品预警

3. 广告数据
   - 今日广告花费
   - 今日广告 ROI

4. 异常情况
   - 订单异常
   - 库存异常
   - 广告异常

5. 行动建议
   - 需要立即处理的事项
   - 优化建议
```

### 周度分析报告

```
请帮我生成周度分析报告：

1. 销售趋势
   - 本周销售数据
   - 与上周对比
   - 趋势分析

2. 商品分析
   - 热销商品
   - 滞销商品
   - 库存预警

3. 广告分析
   - 广告花费趋势
   - 广告效果变化
   - 优化建议

4. 用户分析
   - 新用户数据
   - 复购数据
   - 用户反馈

5. 下周计划
   - 销售目标
   - 商品策略
   - 广告策略
```

### 月度分析报告

```
请帮我生成月度分析报告：

1. 销售总结
   - 本月销售数据
   - 与上月对比
   - 与去年同期对比

2. 商品总结
   - 商品销售排行
   - 库存周转分析
   - 新品表现

3. 广告总结
   - 广告投入产出
   - 各平台广告效果
   - 优化建议

4. 用户总结
   - 用户增长
   - 复购分析
   - 用户反馈

5. 下月计划
   - 销售目标
   - 商品策略
   - 广告策略
   - 用户运营策略
```

## 注意事项

1. **数据质量**：确保数据的准确性和完整性
2. **分析周期**：选择合适的分析周期（日、周、月）
3. **对比基准**：选择合适的对比基准（环比、同比）
4. **行动导向**：分析结果要能指导实际运营决策
