## 订单对比

### 各平台订单数量对比

```
请帮我对比各平台今天的订单情况：
1. 京东订单数量
2. 抖店订单数量
3. 淘宝订单数量
4. 拼多多订单数量
5. 快手订单数量
6. 小红书订单数量
7. 微信小店订单数量
```

### 各平台订单金额对比

```
请帮我对比各平台最近 7 天的订单金额：
1. 各平台总销售额
2. 各平台平均订单金额
3. 各平台订单金额分布
```

### 各平台订单状态对比

```
请帮我对比各平台的订单状态分布：
- 待付款订单数量
- 待发货订单数量
- 已发货订单数量
- 已完成订单数量
- 退款/售后订单数量
```

### 各平台订单时效对比

```
请帮我分析各平台的订单处理时效：
- 平均发货时间
- 平均签收时间
- 超时未发货订单数量
```

## 综合对比

## 对比分析模板

### 日报模板

```
请帮我生成今日各平台经营日报：

1. 销售数据
   - 各平台订单数量
   - 各平台销售额
   - 与昨日对比

2. 库存数据
   - 低库存预警商品
   - 滞销商品预警

3. 广告数据
   - 各平台广告花费
   - 广告 ROI

4. 异常情况
   - 订单异常
   - 库存异常
   - 广告异常
```

### 周报模板

```
请帮我生成本周各平台经营周报：

1. 销售趋势
   - 每日订单数量变化
   - 每日销售额变化
   - 与上周对比

2. 商品分析
   - 热销商品 TOP 10
   - 滞销商品
   - 库存预警

3. 广告分析
   - 广告花费趋势
   - 广告效果变化
   - 优化建议

4. 下周计划
   - 补货计划
   - 促销计划
   - 广告调整计划
```

### 月报模板

```
请帮我生成本月各平台经营月报：

1. 销售总结
   - 各平台销售数据
   - 与上月对比
   - 与去年同期对比

2. 商品总结
   - 商品销售排行
   - 库存周转分析
   - 新品表现

3. 广告总结
   - 广告投入产出
   - 各平台广告效果
   - 优化建议

4. 下月计划
   - 销售目标
   - 商品策略
   - 广告策略
```

## 注意事项

1. **数据一致性**：不同平台的数据统计口径可能不同
2. **时间同步**：确保各平台查询的时间范围一致
3. **权限检查**：确保有所有平台的查询权限
4. **数据延迟**：各平台数据延迟可能不同
