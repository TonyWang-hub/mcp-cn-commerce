## 常见指标说明

| 指标 | 说明 |
|------|------|
| 展示量 (Impressions) | 广告被展示的次数 |
| 点击量 (Clicks) | 广告被点击的次数 |
| 点击率 (CTR) | 点击量 / 展示量 |
| 花费 (Cost) | 广告消耗的金额 |
| 平均点击单价 (CPC) | 花费 / 点击量 |
| 千次展示费用 (CPM) | 花费 / 展示量 * 1000 |
| 转化量 (Conversions) | 完成目标行为的次数 |
| 转化率 (CVR) | 转化量 / 点击量 |
| 投资回报率 (ROI) | 收入 / 花费 |

---

## 注意事项

1. **数据延迟**：广告数据通常有 1-2 小时的延迟
2. **归因窗口**：转化数据可能因归因窗口而变化
3. **时间范围**：查询时建议指定明确的时间范围
4. **权限控制**：确保 API 密钥有查询广告数据的权限
