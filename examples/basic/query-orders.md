## 京东订单查询

### 查询最近 7 天订单

```
请帮我查询京东最近 7 天的订单，按时间倒序排列
```

AI 会调用 `get_order_list` 工具，传入适当的时间范围。

### 按状态筛选订单

```
请帮我查询京东状态为"待发货"的订单
```

常见订单状态：
- `WAIT_SELLER_STOCK_OUT` - 待发货
- `WAIT_GOODS_RECEIVE_CONFIRM` - 已发货，待确认收货
- `FINISHED_L` - 已完成
- `TRADE_CANCELED` - 已取消

### 查询大额订单

```
请帮我查询京东订单金额超过 1000 元的订单
```

### 查询特定商品的订单

```
请帮我查询京东商品 ID 为 123456 的所有订单
```

## 注意事项

1. **时间格式**：不同平台的时间格式可能不同，请参考各平台的 API 文档
2. **分页查询**：数据量大时建议使用分页，避免一次性加载过多数据
3. **状态筛选**：使用状态筛选可以快速定位需要处理的订单
4. **权限控制**：确保 API 密钥有查询订单的权限
