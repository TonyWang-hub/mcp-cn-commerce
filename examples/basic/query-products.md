## 库存分析

### 全平台库存概览

```
请帮我分析各平台的库存情况：
1. 各平台商品总数
2. 各平台库存总量
3. 各平台低库存商品数量（库存 < 10）
```

### 库存预警

```
请帮我查询各平台库存低于 10 的商品，列出：
- 商品名称
- 当前库存
- 最近 7 天销量
- 预计可售天数
```

### 库存周转分析

```
请帮我分析各平台的库存周转情况：
1. 库存周转率
2. 滞销商品（30 天无销量）
3. 畅销商品（7 天销量 > 100）
```

### 库存补货建议

```
请根据以下条件给出补货建议：
1. 库存低于安全库存（10 件）
2. 最近 7 天有销量
3. 预计可售天数 < 7 天
```

## 注意事项

1. **商品 ID**：不同平台的商品 ID 格式可能不同
2. **库存同步**：库存数据可能存在延迟，建议定期查询
3. **价格波动**：商品价格可能因活动、优惠券等原因变化
4. **权限控制**：确保 API 密钥有查询商品的权限
